Resume AI Enhancer - Snapshot
============================

This folder is a copy of the project with all recent changes saved.

Setup:
  1. Open a terminal in this folder.
  2. Run: npm install
  3. Create a .env.local file with your settings, for example:
       ENABLE_LLM=true
       LLM_PROVIDER=openrouter
       OPENROUTER_API_KEY=your-key-here
       OPENROUTER_MODEL=openai/gpt-4o-mini
  4. Run: npm run dev

Your original .env.local was not copied (for security). Add it manually.

You can zip this folder to share or archive it.
